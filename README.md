# expressexpo
 
### Steps to setup and run the server

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/AbdallahIsmaili/expressexpo.git
   cd expressexpo
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. start the server :
   ```bash
   npm start
   ```
3. how to run tests
    ```bash
   npm run test
   ```
